Design Approach

Proximal Policy Optimization

ðŸ’¥ The PPO algorithm uses Policy network, a Deep Neural Network (DNN, as shown in the figure) in order to optimize the objective function by running a number of epochs of Gradient descent over mini batches in order to give optimal Policy as output.

ðŸ’¥ Another great advantage of PPO is that it avoids large Policy updates by using a ratio that shows the difference between the Old vs New policy, thus improving performance and training experience.

Curriculum Learning

ðŸ’¥ Training of deep learning networks can be facilitated by using information gained from past related undertakings. There are a few procedures including transfer learning, multitask learning and curriculum learning.

ðŸ’¥ We center around curriculum learning since it can start learning in test systems with visual rendering and afterward the trained model can be calibrated in true applications.

ðŸ’¥ Implemented Curriculum learning to improve the learning experience of the agent which is evident from peak Total Cumulative Rewards achieved is shorter training period.

LSTM

ðŸ’¥ When the agent encounters concave obstacles (cul-de-sacs) a lack of memory is a problem. The PPO agent steps back and forth or circles repeatedly as it cannot remember previous movements. To overcome this, we added an LSTM memory layer for the PPO deep learning,

ðŸ’¥ An LSTM is a gated recurrent neural network capable of learning longer-term dependencies making it ideal to provide a memory layer for the agent.
